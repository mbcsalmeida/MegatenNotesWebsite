Megaten Games Checklist

A list of all the megaten games I've speedrun, including ones I haven't made notes for.

If a game has a remake or a re-release it's likely I'll only ever run one of the versions of the game, regardless of how different they may be (for example: P3, P3FES and P3P).

There's a checkmark for every game run, a pentagram for every remake/remaster I will therefore never run and a cross for every game I frankly consider un-speedrun-able.
