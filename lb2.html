Bugs and Features

MP-less Special Attack

In order to use a character’s ”Special Attack” during a fight, press the ”select” button while choosing Attack or Auto. This will use a given character’s strongest attack, including magic spells, without using any MP.

Pon & Monster

If you name the protagonist "I'm Pon" in japanese "ポンです", presumably a dev easter egg, you'll start your playthrough with extremely high stats and breeze through the entire game.

If you name the protagonist "Monster" in japanese "モンスター" you'll gain access to the full demon compendium and not start a playthrough.

Glitched demons

If you leava a demon in the demon trainer, which levels up your demons for you, as you go pick it back up, if you do so while holding select, you'll receive a glitched demon instead with extremely high stats. As this is a glitch, it's hard to access if using it would cause harm to the savefile or weird things from happening during battles.
