Megami Tensei Gaiden: Last Bible I

About

Timer starts on START and ends on the last frame of the game.

Demon recruitments are based on the demon’s type. Knowing the type and the initial dialogue line with which they initiate the conversation allows us to be able to know how to recruit them immediately as recruitments are based on fixed decision trees. Talking to the demons using anyone other than Hero leads to different results, however, and it is not recommended.

There is a demon negoation section at the end of the notes for information on recruitments.

Demon Negotiation

These are the three possible starting dialogue lines that define each decision tree. Knowing the demon’s type, follow the correct order of answers.

”Are you the one going around killing monsters?” / ” なんだきさま ”

  • Type 1: Yes, Yes, No, No.
  • Type 2: No, Yes, Yes, Yes.
  • Type 3: Yes, Yes, Yes, Yes.
  • Type 4: Yes, Yes, Yes, No.
  • Type 5: No, Yes, No, No.

”Do you want me to join you?” / ” わたしのなかま ”

  • Type 1: No, No, Yes, No.
  • Type 2: Yes, Yes, Yes, Yes.
  • Type 3: No, No, Yes, Yes.
  • Type 4: No, No, No, No.
  • Type 5: Yes, Yes, Yes, No.

”I’m saved! If you wish to speak with me, would you please lower your weapon?”/” ホイ ”

  • Type 1: No, Yes, No, Yes.
  • Type 2: Yes, No, Yes, No.
  • Type 3: Yes, Yes, Yes, Yes.
  • Type 4: Yes, No, Yes, Yes.
  • Type 5: Yes, No, No, Yes.
